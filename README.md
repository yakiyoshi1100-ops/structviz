# StructViz — コンサルティング構造化ビジュアライザー

## 概要
会議中にテキストを入力し、コンサルフレームワークにリアルタイムで当てはめて可視化するWebアプリ。話の発散を防ぎ、クライアントとファシリテーターが同じ構造を見ながら議論を進められるようにする。

## ローカル起動手順
```bash
cd structviz
npm install
npm run dev
# ブラウザで http://127.0.0.1:5173 を開く
```

## 分析モードの選択
起動時に以下の2モードを選択する：

### スタンドアロンモード（ルールベース）
- APIキー不要、オフライン動作可
- キーワードマッチングで即時分類（レイテンシ0）
- **推奨入力：フレームワーク固有のキーワードを含む短文（1〜5文）**
- **注意：自然文の長文や文脈理解が必要な分類はAIモードを使用すること**

#### スタンドアロンモードで精度が出るフレームワーク
| フレームワーク | 有効なキーワード例 |
|---|---|
| Whyツリー | 「下がっている」「原因」「要因」「モチベーション」|
| SWOT分析 | 「強み」「弱み」「機会」「脅威」|
| 3C分析 | 「顧客」「競合」「自社」|
| PEST分析 | 「政治」「経済」「社会」「技術」「AI」|
| 演繹法 | 「一般に」「よって」「したがって」|

#### スタンドアロンモードの限界
以下のフレームワークは文脈理解が必要なため、スタンドアロンでは精度が低い。AIモードを推奨：
- As-Is/To-Be/Gap（現状・理想・差分の文脈判断が必要）
- Howツリー（手段・方法を示す文脈判断が必要）
- カスタマージャーニーマップ
- バリューストリームマッピング
- DMAIC

### AIモード（Claude API）
- Claude APIキーが必要
- 自然文を文脈理解して高精度に分類
- 全33フレームワークに対応
- インターネット接続必要

#### APIキーの取得
1. https://console.anthropic.com にアクセス
2. アカウント作成・ログイン
3. API Keys → Create Key
4. 生成されたキー（sk-ant-...）をコピー
5. StructVizの起動画面でAIモードを選択し貼り付け

**注意：APIキーはブラウザのlocalStorageにのみ保存。外部サーバーへの送信はClaude API呼び出し時のみ。**

## 操作ガイド

### ファシリテーターモード
- 左サイドバー：フレームワーク選択
- 下部入力パネル：テキスト入力・構造化ボタン
- 中央キャンバス：フレームワーク図（ドラッグで移動、ダブルクリックで編集）
- 左下：未分類バッファ（分類できなかったテキストを手動で配置可）

### プレゼンテーションモード
- ヘッダーの「presentation」ボタン または F11 で切り替え
- フレームワーク図が全画面表示
- 右下のフローティングパネルからいつでも追加入力可能

### ショートカット
| キー | 動作 |
|---|---|
| F11 | ファシリテーター/プレゼンモード切り替え |
| Ctrl+Enter / Cmd+Enter | 構造化実行 |
| Escape | 設定パネルを閉じる |

### エクスポート
ヘッダーのエクスポートメニューから以下の形式でダウンロード可能：
- PNG：フレームワーク図のスクリーンショット
- Markdown：アウトライン形式のテキスト
- JSON：構造データ（再インポート可能）
- SVG：Phase 2対応予定

## 音声ファイル文字起こし（Phase 3）

### 対応形式
.mp3 / .wav / .m4a / .webm / .mp4

### Whisperモードの選択（設定パネルから）

**OpenAI Whisper API（推奨）**
1. https://platform.openai.com でAPIキーを取得
2. 設定パネル → 音声文字起こし → OpenAI Whisper APIを選択
3. OpenAI APIキー（sk-...）を入力して保存

**ローカルWhisper**
1. faster-whisper-server をローカルで起動
   参考: https://github.com/fedirz/faster-whisper-server
2. 設定パネル → ローカルWhisperを選択
3. エンドポイントURL（デフォルト: http://localhost:9000/asr）を確認して保存

### 使い方
1. 入力パネルの「📁 ファイル」タブを選択
2. 音声ファイルをドロップまたはクリックして選択
3. 「文字起こし開始」ボタンを押す
4. 完了後、テキストが入力欄に自動入力される
5. フレームワークを選択して「構造化」ボタンを押す
